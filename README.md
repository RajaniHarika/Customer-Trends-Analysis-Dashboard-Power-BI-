# Customer-Trends-Analysis-Dashboard-Power-BI-
The report includes visualizations such as sales trends, customer distribution, and performance metrics
This project presents an interactive Customer Trends Analysis Dashboard built using Microsoft Power BI.
The dashboard analyzes customer data to identify trends, patterns, and insights that help businesses make better decisions.

The report includes visualizations such as sales trends, customer distribution, and performance metrics.
